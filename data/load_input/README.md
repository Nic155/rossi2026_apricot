# load_input

The deleterious-variant table is not stored in this repository due to its size
(~1.3 GB). It is produced by the upstream genetic-load pipeline
**WP5_genetic_load** (https://github.com/Nic155/WP5_genetic_load).

Download it from the data deposit and place it in this folder:

- Recherche Data Gouv, DOI 10.57745/D1LNG3 (https://doi.org/10.57745/D1LNG3)
- File: `result3-provean_W2_C1_subset.tsv`

This table is the main input consumed by the `roh_load`, `sweep_burden`, and
`R_analysis` stages.
